# Projeto de Comunicação Cliente-Servidor via Sockets

## 🎯 Objetivo Geral
- Desenvolver uma aplicação baseada na arquitetura cliente-servidor capaz de fornecer transporte confiável de dados na camada de aplicação. O sistema simula o controle de fluxo e erro em um canal de comunicação sujeito a perdas, utilizando protocolos de retransmissão como Go-Back-N (GBN) ou Selective Repeat (SR).
